1. 파세리가 있는 내 계정을 이용하는 경우

  • 설정에서 내 코나미 ID/pass, iidx.me ID/pass(선택)를 입력합니다.
  • 이아무세 사이트를 들어간 뒤 마우스 오른쪽 버튼을 눌러 메뉴를 확인합니다.
  • 다음과 같은 방식으로 갱신하시면 됩니다: 내 계정 로그인 - 투덱미 갱신

2. 파세리가 있는 타인의 계정을 이용하는 경우

  • 설정에서 내 코나미 ID/pass, 주인님 코나미 ID/pass, 내 이어뮤 ID/pass, 주인님 이어뮤 ID(선택), iidx.me ID/pass(선택)를 입력합니다.
  • 이아무세 사이트를 들어간 뒤 마우스 오른쪽 버튼을 눌러 메뉴를 확인합니다.
  • 다음과 같은 방식으로 갱신하시면 됩니다: 내 계정 로그인 - 이어뮤 제거 - 주인님 계정 로그인 - 내 카드 추가 - 투덱미 갱신 - 내 카드 제거 - 내 계정 로그인 - 내 카드 추가
  • 주인님 카드 ID를 입력하시면, 내 이어뮤를 제거한 뒤 주인님 이어뮤를 쉽게 사용 상태로 만들 수 있습니다. (메뉴 기능 사용 필요)
  • iidx.me id/pass는 반드시 입력하지 않아도 되나, 입력하면 투덱미 갱신 스크립트 적용 후 아이디 비번이 자동으로 입력됩니다.
